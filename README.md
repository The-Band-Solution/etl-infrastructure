# Immigrant ETL Infraestruture

## 🚀 Goal

Provides a etl infrastructure to retrieve data from applications. 

## ⚙️ Requirements

1. Docker Compose

## 🔧 Install

In a terminal execute the following commnad to start the infrastructure

```bash
docker-compose up
```
